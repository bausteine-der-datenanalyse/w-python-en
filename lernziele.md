In this module, the fundamentals of programming with Python are introduced. In this module, you will learn to...

- Become familiar with the basic concepts of object-oriented programming.

- Write Python code, create variables, perform operations, and format output.

- Read and use documentation.

- Understand Python's exclusive counting system (zero-based indexing).

- Know the difference between functions and methods and how to write your own functions.

- Load modules and packages.

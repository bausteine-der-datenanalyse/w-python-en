5  Collection Types

Collection types are used to store and manage multiple values in a single variable. In Python, there are four collection types, each represented by its own class:

  • Lists can contain a flexible number of elements of any type.

  • Tuples can also contain elements of any type, but they are immutable.

  • Sets are unordered collections that can contain each element only once.

  • Associative Arrays, or dictionaries, are key-value mappings.

In this chapter, the four collection types are briefly introduced first. Then, we will focus on how to work with lists in particular.

5.1 Lists

Like all types in Python, lists are created by assignment. When defining a list, elements are enclosed in square brackets [] and separated by commas ,. Lists can be concatenated with the + operator. * repeats a list n times.

text_variable = 'abc'

list1 = [1, 'xy', True, text_variable]
print(list1)

# Lists can also contain other lists
list2 = [None, list1]
print(list2)

# Lists can be concatenated with + and repeated with *
print(list1 + list2)
print(list1 * 2)
[1, 'xy', True, 'abc']
[None, [1, 'xy', True, 'abc']]
[1, 'xy', True, 'abc', None, [1, 'xy', True, 'abc']]
[1, 'xy', True, 'abc', 1, 'xy', True, 'abc']

An empty list can be created by assigning [].

empty_list = []
print(empty_list)
[]

Slicing: the Access Operator []

Accessing individual or multiple elements of a list (and other collection types) is done using the access operator []. A portion of an object is called a slice, which is why the operator is also called the slice operator.

Accessing Individual Elements

Elements are accessed by their index, starting at 0.

print(list1)
print(list1[0])
print(list1[3])
[1, 'xy', True, 'abc']
1
abc

Nested lists can be accessed using two consecutive indexing operators. The list list2 contains, at index position 1, a list with 4 elements.

print(list2)
print(list2[1])
print(list2[1][0], list2[1][1], list2[1][2], list2[1][3])
[None, [1, 'xy', True, 'abc']]
[1, 'xy', True, 'abc']
1 xy True abc

With negative indices, elements can be accessed from the end of a list. For example, -1 refers to the last element.

print(list1)
print(list1[-1], list1[-3])
[1, 'xy', True, 'abc']
abc xy

Accessing Multiple Elements

Index ranges can be accessed using the [start:stop:step] syntax. start is the first element to be included, stop is the first element not included, and step defines the increment.

Access Operator Slice
list[start:stop] Elements from start to stop - 1
list[:] All elements of the list
list[start:] Elements from start to the end of the list
list[:stop] Elements from the beginning of the list up to stop - 1
list[::3] Every third element

 

Negative values for start, stop, or step select elements in reverse order.

Access Operator Slice
list[-1] the last element of the list
list[-2:] the last two elements of the list
list[:-2] all elements except the last two
list[::-1] all elements in reverse order
list[1::-1] the first two elements in reverse order
list[:-3:-1] the last two elements in reverse order
list[-3::-1] all elements except the last two in reverse order

List Methods

Various methods are defined for the list type.

Determining Elements

  • list.index(x, start, stop) returns the index position of the first element x. The optional arguments start and stop allow you to limit the search range.

  • list.count(x) returns the frequency of x in the list.

  • list.reverse() reverses the order of the list elements (this modifies the list!).

  • list.sort(reverse = False) sorts the list, with the optional argument reverse = True for descending order (this modifies the list!). The data types within the list must be sortable (i.e., all elements are numeric or strings).

print(list1)

list1.reverse()
print(list1)

# True is counted as 1
print("True is counted as 1:", list1.index(1), list1.count(1))
[1, 'xy', True, 'abc']
['abc', True, 'xy', 1]
True is counted as 1: 1 2

Inserting Elements

  • list.append(x) adds a single element to the end of the list.

  • list.extend(iterable) adds all elements from the given iterable to the end of the list. The iterable can be a list, a tuple, a set, or a dictionary.

  • list.insert(i, x) inserts element x at position i.

print(list1, "\n")

list1.append('Hello')
list1.extend(['Hello', 'World!'])
list1.insert(2, '12345')

print(list1)
['abc', True, 'xy', 1] 

['abc', True, '12345', 'xy', 1, 'Hello', 'Hello', 'World!']

Removing Elements

  • list.remove(x) removes the first occurrence of x in the list and raises a ValueError if x is not in the list.

  • list.pop(i) removes the element at index position i. If no index is specified, the last element is removed. The method list.pop(i) returns the removed element.

  • list.clear() removes all elements from a list.

list1.remove('Hello')
print(list1)

list1.pop(2)
['abc', True, '12345', 'xy', 1, 'Hello', 'World!']
'12345'

Copying Lists and List Elements

In Python, lists do not store data directly; instead, they consist of references (pointers) to the memory locations of the elements they contain. When a list is created by assigning it to another list, the elements themselves are not copied; instead, both lists refer to the same memory locations.

# Copying by assignment
list1 = [1, 'xy', True, text_variable]
print("list1:", list1, "\n")
list2 = list1

## Changing an element in list2
list2[0] = 'ABC'
print("list1 also changes because of the assignment to list2:", list1, "\n")
list1: [1, 'xy', True, 'abc'] 

list1 also changes because of the assignment to list2: ['ABC', 'xy', True, 'abc']

To copy a list and create a new object in memory, the method list.copy() can be used. Using the slice operator [:] also creates a new list in memory.

# Using the list.copy() method
list1 = [1, 'xy', True, text_variable]
list2 = list1.copy()

# Modify an element in list2
list2[0] = 'ABC'
print("list1 remains unchanged by the assignment to list2:", list1, "\n")

# Using the slice operator
list1 = [1, 'xy', True, text_variable]
list2 = list1[:]

# Modify an element in list2
list2[0] = 'ABC'
print("list1 remains unchanged by the assignment to list2:", list1)
list1 remains unchanged by the assignment to list2: [1, 'xy', True, 'abc'] 

list1 remains unchanged by the assignment to list2: [1, 'xy', True, 'abc']

Copying list items is not a problem in this regard.

# Using the slice operator
list1 = [1, 'xy', True, text_variable]
list2 = list1[0:2]

# Changing an element in list2
list2[0] = 'ABC'
print("list1 remains unchanged by the assignment in list2:", list1)
list1 remains unchanged by the assignment in list2: [1, 'xy', True, 'abc']

To check whether two objects share the same memory space, you can compare their object IDs using the id() function or use the is / is not operators, which internally call id().

list1 = [1, 'xy', True, text_variable]
list2 = list1

print("ID of list1:", id(list1))
print("ID of list2:", id(list2))
print("Is ID of list1 equal to ID of list2:", list1 is list2)
ID of list1: 1498764172096
ID of list2: 1498764172096
Is ID of list1 equal to ID of list2: True
Hinweis 5.1: Identity vs. Value Equality

The is operator checks the identity of two objects and differs from the logical operator ==, which checks for value equality. Since list1 and list2 contain the same elements, both value equality and identity are relevant. The difference between identity and value equality can be illustrated with a single value (the code example suppresses a syntax warning).

# Value equality
print(1 == 1.0)
print(list1 == list2, "\n")

# Identity
print(1 is 1.0)
print(list1 is list2)
True
True 

False

5.2 Tuples

Tuples are very similar to lists, but tuples are immutable data objects. This means that the elements of an existing tuple cannot be changed or removed. You also cannot add new elements to a tuple.

Tuples are created using parentheses (), and elements are separated by commas ,. A tuple with a single value is created with a comma in the form (value, ). Accessing elements of a tuple is possible using the slice operator [start:stop:step]. Tuples can be concatenated using the + and * operators.

tuple1 = (2, 7.8, 'Fire', True, text_variable)
tuple2 = (1, )

print(tuple1)
print(tuple1[2:4])
print(tuple1[::2])
print(tuple1[-1])
print(tuple1[2:4] + tuple2)
print(3 * tuple2)
(2, 7.8, 'Fire', True, 'abc')
('Fire', True)
(2, 'Fire', 'abc')
abc
('Fire', True, 1)
(1, 1, 1)

Copying Tuples

Tuples behave differently from lists when it comes to copying. Tuples do not have a .copy() method. Using slicing [:] allows you to create a copy, but both tuples initially reference the same memory. When a tuple is reassigned, Python, as with any object, creates a new object in memory.

# Copying by assignment
tuple1 = (1, 2, 3)
tuple2 = tuple1

## Reassigning values of tuple1
tuple1 = (4, 5, 6)
print(f"The values stored in tuple2 remain unchanged:\n{tuple1} {tuple2}\n")

# Copying using slice operator
tuple1 = (1, 2, 3)
tuple2 = tuple1[:]
print(tuple2 is tuple1)

## Reassigning values of tuple1
tuple1 = (4, 5, 6)
print(tuple1, tuple2)
The values stored in tuple2 remain unchanged:
(4, 5, 6) (1, 2, 3)

True
(4, 5, 6) (1, 2, 3)

5.3 Sets

In Python, sets can be created from a list or a tuple using the set() function, or directly with curly braces {} (an empty set can only be created with set(), because {} creates an empty dictionary). Sets are unordered collections, so their elements do not have a specific order.

my_list = [1, 1, 5, 3, 3, 4, 2, 'a', 123, 1000, ('tuple', 5)]
print("The list object as a set:\n", set(my_list))

my_set = {1, 2, 3, 4, 5, 1000, ('tuple', 5), 'a', 123}
print("A set can also be created using curly braces:", my_set, sep="\n")
The list object as a set:
 {1, 2, 3, 4, 5, 1000, 'a', ('tuple', 5), 123}
A set can also be created using curly braces:
{1, 2, 3, 4, 5, 1000, ('tuple', 5), 'a', 123}

Sets can be used, for example, for comparison operations.

set_a = set('Python')
set_b = set('is awesome')

# unique characters in a
print("Set a:", set_a)

# characters in a but not in b
print("Set a - b:", set_a - set_b)

# characters in a or b
print("Set a | b:", set_a | set_b)

# characters in a and b
print("Set a & b:", set_a & set_b)

# characters in a or b, but not in both (XOR)
print("Set a ^ b:", set_a ^ set_b)
Set a: {'t', 'y', 'P', 'n', 'o', 'h'}
Set a - b: {'t', 'P', 'y', 'n', 'h'}
Set a | b: {'t', 'm', ' ', 'P', 'y', 'w', 's', 'e', 'n', 'o', 'h', 'a', 'i'}
Set a & b: {'o'}
Set a ^ b: {'m', 't', ' ', 'y', 'P', 'w', 's', 'e', 'n', 'a', 'h', 'i'}

Copying Sets

Sets behave similarly to tuples, with the difference that the .copy() method is defined for sets. However, the indexing operator [] cannot be used on sets.

# Copying by assignment
set1 = {1, 2, 3}
set2 = set1
print(set1 is set2)

## Reassigning set1
set1 = {4, 5, 6}
print(f"The values stored in set2 remain unchanged:\n{set1} {set2}")

# Copying using the .copy() method
set1 = {1, 2, 3}
set2 = set1.copy()
print(set1 is set2)

## Reassigning set1
set1 = {4, 5, 6}
print(f"The values stored in set2 remain unchanged:\n{set1} {set2}")

5.4 Dictionaries

Dictionaries consist of key-value pairs. Keys can be numbers or strings, and each key can only appear once. Dictionaries are defined using curly braces {}. Keys and their corresponding values are separated by a colon :. Values are accessed using the bracket operator [], which contains the key(s). Accessing values by the positional index of keys is not possible, since numbers are interpreted as keys.

dictionary1 = {1: 'abc', 'b': [1, 2, 3], 'c': ('tuple', 5, 6)}
print(dictionary1, "\n")

print("Values of key 1:", dictionary1[1])
print("Values of key 'b':", dictionary1['b'])
{1: 'abc', 'b': [1, 2, 3], 'c': ('tuple', 5, 6)} 

Values of key 1: abc
Values of key 'b': [1, 2, 3]

You can access the keys of a dictionary using the dictionary.keys() method, and the values using the dictionary.values() method.

print("Keys:", dictionary1.keys(), "\n")
print("Values:", dictionary1.values())
Keys: dict_keys([1, 'b', 'c']) 

Values: dict_values(['abc', [1, 2, 3], ('tuple', 5, 6)])

Copying Dictionaries

Dictionaries behave like lists when copied, meaning that copying via assignment will share the same memory space between dictionaries.

# Copying via assignment
print("dictionary:", dictionary1, "\n")
dictionary2 = dictionary1

## Modifying an element in dictionary2
dictionary2[1] = 'ABC'
print("dictionary1 also changes due to the assignment to dictionary2:\n",
       dictionary1, "\n")

# Using the dictionary.copy() method
dictionary1 = {1: 'abc', 'b': [1, 2, 3], 'c': ('tuple', 5, 6)}
dictionary2 = dictionary1.copy()

## Modifying an element in dictionary2
dictionary2[1] = 'ABC'
print("dictionary1 remains unchanged by the assignment to dictionary2:\n",
       dictionary1, "\n")

5.6 Deleting: the del Keyword

To delete collections, elements, or slices, the del keyword can be used.

# Deleting a list
del list1

# Deleting a range of indices from a list
print("List before deletion:", list2)
del list2[1:3]
print("List after deletion:", list2)

# Deleting a key from a dictionary
print("Dictionary before deletion:", dictionary1)
del dictionary1[1]
print("Dictionary after deletion:", dictionary1)
List before deletion: [1, 'xy', True, 'abc']
List after deletion: [1, 'abc']
Dictionary before deletion: {1: 'abc', 'b': [1, 2, 3], 'c': ('tuple', 5, 6)}
Dictionary after deletion: {'b': [1, 2, 3], 'c': ('tuple', 5, 6)}

5.7 Functions

Collection types can be converted into each other.

dictionary = {1: 'Tomcat', 2: 'Vixen', 3: 'Doe'}
print( (list_var := list(dictionary)) )
print( (set_var := set(list_var)) )
print( (tuple_var := tuple(set_var)) )
[1, 2, 3]
{1, 2, 3}
(1, 2, 3)

Some practical functions can also be applied to collection types:

  • len() returns the number of elements in a collection.
  • min(), max(), sum() return the minimum, maximum, or sum of a collection (for dictionaries, the number of keys is counted).
